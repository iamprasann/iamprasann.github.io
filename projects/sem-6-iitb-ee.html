SEM-6-IITB-EE

SEM-6-IITB-EE is a semester repository for Semester 6 coursework at IIT Bombay Electrical Engineering. At first pass, it looks more advanced and more project-oriented than the earlier semester repos: the course set spans operating systems, hardware design, technical communication, radar systems, communication networks, and machine learning, and several folders preserve both source material and final deliverables.

Overview

  • Repository: iamprasann/SEM-6-IITB-EE
  • Created: September 6, 2023
  • Updated: September 6, 2023
  • Primary language on GitHub: Jupyter Notebook

Top-level course folders:

  • CS347M - Operating Systems
  • EE344 - Electronic Design Lab
  • EE350 - Technical Communication
  • EE602 - Radar Systems
  • EE706 - Communication Networks
  • EE769 - Machine Learning

The root README is minimal, so the semester narrative has to be reconstructed from the artifact structure and file names.

Why It Mattered

This repository matters because it captures a semester where the work appears to broaden in both technical range and output format. Instead of mainly weekly lab sheets or assignment bundles, the repo shows:

  • systems work with reports and reference material
  • hardware design documentation and proposal/final-report structure
  • seminar-style communication deliverables
  • a radar project with source readings and a typeset report
  • machine learning notebooks, scripts, data, and personal notes

That mix makes the repo valuable not only as a semester archive but also as a source for later concept and course pages.

How It Worked

Each course folder has its own mode of work.

  • CS347M - Operating Systems mixes readings, PDFs, images, a programming-assignment folder, and a LaTeX report.
  • EE344 - Electronic Design Lab preserves proposal material, block diagrams, circuit design files, and a final-report asset folder centered on buck-converter-related components.
  • EE350 - Technical Communication preserves seminar materials, including an organic transistors report and presentation.
  • EE602 - Radar Systems is organized around a project and presentation folder with multiple SAR references and a nested report directory containing sar.tex and sar.pdf.
  • EE706 - Communication Networks stores project PDFs directly at the course root.
  • EE769 - Machine Learning combines assignments with .ipynb, .py, and .csv files alongside weekly course notes in PDF form.

The clearest first-pass themes are breadth and documentation. This repo is not just storing answers; it is storing a working record of reading, design, implementation, and communication across very different technical domains.

Lessons Learned

On a first pass, SEM-6-IITB-EE looks like a transition point in the undergraduate record. The courses are more diverse, the artifacts are more mature, and the outputs show stronger signs of project framing rather than only assignment completion.

The main follow-on opportunities for a later pass are clear:

  • a dedicated page for the EDL hardware project
  • a dedicated page for the SAR project in EE602
  • course pages for CS347M, EE344, EE602, and EE769
