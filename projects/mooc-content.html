MOOC-content

MOOC-content is a self-study repository that archives practice and assignments from major online courses rather than institutional coursework. It combines algorithmic programming assignments, the full Andrew Ng machine-learning exercise sequence, and a Deep Learning Specialization completion artifact. In the knowledge base, it reads as a record of structured learning outside the formal undergraduate curriculum.

Overview

  • Repository: iamprasann/MOOC-content
  • Created: July 21, 2021
  • Updated: July 21, 2021
  • Primary language on GitHub: MATLAB

Top-level structure:

  • DSA Practice Assignments
  • Machine Learning - Andrew NG
  • Deep Learning Specialization Certificate.pdf

The root README explicitly links the Coursera sources, so the repo already explains itself better than many of the other archival repositories.

Why It Mattered

This repository matters because it captures the self-directed learning layer behind the rest of the undergraduate work. It shows deliberate work on:

  • data structures and algorithms implementation practice in C++
  • classical machine-learning assignments in MATLAB/Octave
  • the broader deep-learning learning path, at least at the certificate/completion level

That makes it useful evidence of preparation and breadth outside formal course repositories.

How It Worked

The repo is organized by MOOC track rather than by semester or project.

  • DSA Practice Assignments contains three major specialization folders: Algorithmic ToolBox, Algorithms on Graphs, and Data Structures.
  • The recursive file listing shows week-by-week PDFs plus many C++ solutions spanning warmup problems, greedy algorithms, divide-and-conquer, dynamic programming, graph decomposition, shortest paths, MST, hash tables, BSTs, and related problems.
  • Machine Learning - Andrew NG contains machine-learning-ex1 through machine-learning-ex8, which matches the full exercise sequence of the classic course.
  • Those folders preserve the standard assignment PDFs, datasets, helper code, and .m solutions for topics like regression, classification, neural networks, bias/variance, SVMs, clustering, PCA, anomaly detection, and recommender systems.
  • The root-level deep-learning certificate PDF preserves completion evidence for the Deep Learning Specialization even though the underlying assignment code is not stored here.

Taken together, the repo looks like a disciplined archive of online-course execution rather than a single end-to-end project.

Key Artifacts

  • Deep Learning Specialization Certificate.pdf

Representative artifacts visible in the repo include:

  • weekly DSA assignment PDFs and C++ implementations
  • eight MATLAB/Octave machine-learning exercise sets with datasets and helper files
  • a deep-learning specialization certificate

Lessons Learned

On a first pass, MOOC-content reads as a strong support repository rather than a narrative project repo. Its value comes from what it demonstrates: systematic practice, breadth across algorithms and ML, and external learning that likely fed into later coursework and projects.

The clearest second-pass opportunities are:

  • a concept page linking the DSA practice to later systems or algorithm-heavy work
  • a concept page linking the Andrew Ng ML exercises to later machine-learning projects in the corpus
