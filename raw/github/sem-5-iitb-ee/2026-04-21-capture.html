Course-Level Notes

CS663

  • Contains Assignment 1, Assignment 2, and fingerprint recognition project.
  • Recursive file listing shows many report PDFs and images under assignment folders.
  • README.md
  • CS663-report.pdf

The project README identifies the codebase as Walking-to-SP, a MATLAB implementation of a walking algorithm for fingerprint singular point detection. It cites:

  • Zhu E, Guo X, Yin J. "Walking to singular points of fingerprints." Pattern Recognition, 2016.
  • Guo X, Zhu E, Yin J. "A fast and accurate method for detecting fingerprint reference point." Neural Computing & Applications, 2016.

The README also notes an external dependency on Peter Kovesi's "MATLAB and Octave Functions for Computer Vision and Image Processing".
