# SEM-7-IITB-EE

`SEM-7-IITB-EE` is a compact semester repository that differs sharply from the earlier semester archives. Instead of many course folders, it contains one focused course area, `EE 6103 - Advanced Network Security`, and a separate `placements` directory. That makes it less of a conventional semester log and more of a semester snapshot organized around one advanced academic thread plus career-preparation material.

## Why It Mattered

This repository matters because it captures a semester that looks more selective and more outcome-oriented than the previous ones. The knowledge here is concentrated in two directions:

- a project-based advanced network security course
- placement-preparation documents that sit alongside academic work

That is useful for the knowledge base because it shows not just what was studied, but also how the semester intersected with professional preparation.

## How It Worked

The academic core of the repo is `EE 6103 - Advanced Network Security`. That folder is divided into `Midsem Project` and `Endsem Project`, and both sections preserve reports and presentations. The visible artifacts suggest a course built around substantial project submissions rather than a sequence of small assignments. The midsem project also includes a `Problem_statement.pdf` and an `IITKHackathonPresentation.pptx`, which hints at an external or competition-adjacent component in the course workflow.

The second major branch is `placements`, which contains one-page and two-page technical profile documents in both `.tex` and `.pdf` forms, plus a `proofs` folder with supporting evidence files. This folder preserves a different kind of work: self-presentation, validation, and career preparation rather than coursework proper.
