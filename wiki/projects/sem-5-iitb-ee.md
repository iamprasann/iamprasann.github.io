# SEM-5-IITB-EE

`SEM-5-IITB-EE` is a semester repository for Semester 5 coursework at IIT Bombay Electrical Engineering. Compared with `SEM-4-IITB-EE`, this repo looks less like a broad mix of code and more like a structured archive of reports, lab manuals, prelabs, problem sheets, and course submissions, with one especially notable project nested inside `CS663`.

## Overview

- Repository: [iamprasann/SEM-5-IITB-EE](https://github.com/iamprasann/SEM-5-IITB-EE)
- Created: September 4, 2021
- Updated: November 18, 2021
- Primary language on GitHub: Python
- Top-level course folders: `CS663`, `EE230`, `EE324`, `EE340`

The root README is minimal, so most of the semester story has to be reconstructed from the folder structure and artifact types.

## Why It Mattered

This repository matters because it preserves the working record of a semester in a different mode from the previous one. Here the knowledge is carried less by polished top-level explanations and more by the density of the artifacts themselves:

- assignment reports
- lab writeups
- prelab material
- manuals and problem sheets
- one research-style project with explicit paper references

That makes this repo a strong source for later course pages and for tracing how academic work was documented and submitted.

## How It Worked

The semester is divided into four course-code folders, each with a distinct rhythm.

- `CS663` contains two assignment groups plus a `fingerprint recognition project`.
- `EE230` is organized as `Expt-0` through `Expt-5`, suggesting a lab progression.
- `EE324` is organized as `Lab 1` through `Lab 5`, each paired with problem sheets and submitted PDFs.
- `EE340` runs from `Week 1` through `Week 9`, with repeated prelab and lab sheet patterns.

The most distinctive sub-area is the `CS663/fingerprint recognition project` folder. Its README describes the code as `Walking-to-SP`, a MATLAB implementation for detecting fingerprint singular points. It cites a 2016 Pattern Recognition paper and a related 2016 Neural Computing & Applications paper, and it notes a dependency on Peter Kovesi's MATLAB computer vision functions. That makes this folder the clearest bridge between coursework and research-style implementation in the repository.

## Lessons Learned

On a first pass, this repository shows a more archival style of academic work than `SEM-4-IITB-EE`. The repo is less self-explanatory, but still rich: the combination of repeated lab artifacts and one clearly named project makes it useful both as evidence of coursework volume and as a source for more focused follow-up pages.

The most important next step here is to split the semester outward in two directions:

- course pages for `CS663`, `EE230`, `EE324`, and `EE340`
- a dedicated page for the fingerprint recognition project, which already has a narrative README, a report, and cited references
